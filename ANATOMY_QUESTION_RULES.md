# Anatomy Bank Question Rules (Extremely Hard)

These rules standardize every **Anatomy by Systems** item added to the LVN HESI practice app.

## 1) Filters & metadata
- **Bank:** `Anatomy`
- **Body System (`system`):** one of  
  Cardiovascular, Respiratory, Nervous System, Endocrine, Gastrointestinal, Hepatic, Hematologic, Integumentary, Musculoskeletal, Urinary
- **Topic (`topic`):** short label that helps students *search and drill*.  
  - Must be **meaningful** (e.g., “cranial nerve lesion localize”, “portal hypertension signs”, “anterior compartment syndrome”).
  - Avoid generic tags like “anatomy function”.

## 2) Difficulty
- Single level only: **Extremely Hard**
- No explicit difficulty selector in the UI; question rigor must come from *clinical reasoning + anatomy localization*.

## 3) Allowed question types
### A) Standard (Single best answer)
- Exactly **1 best answer**.
- All distractors must be **plausible** and belong to the same concept category (no “obvious throwaways”).
- Include *at least one* distractor that is correct in a **different** clinical context (classic trap).

### B) SATA (Select all that apply)
- **2–4 correct** answers (avoid 1 or all 5 unless there is a strong reason).
- Each option must be independently true/false (no double‑barreled options).
- Avoid “always/never” unless it is a true absolute.

### C) Bowtie
- Must test **clinical anatomy decision‑making** (localization → consequence → action).
- App structure uses **3 columns**:
  - Left = diagnosis/structure at risk/localization
  - Middle = priority action / assessment focus
  - Right = outcome/next step (if not applicable, use `N/A`)
- Correct answers must be unambiguous and supported by the stem.

## 4) Stem design (what makes it “hard”)
Every stem should include at least **two** of:
- an anatomic landmark (foramen, canal, compartment, space)
- a *pattern* (crossed findings, dermatomal distribution, reflex level, gait)
- a mechanism (ischemia, compression, transection, herniation, obstruction)
- a nursing‑relevant red flag (airway risk, shock, neuro decline, compartment syndrome)

## 5) Rationales
- Provide **1–3 sentences** explaining:
  1) why the correct answer is correct  
  2) why the main distractor is wrong (or why the others are less best)
- If the question is a “don’t miss” emergency pattern, the rationale must state the **urgent nursing priority**.

## 6) Quality checks (must pass)
- Answer key matches the displayed choices (no mismatches).
- No duplicate stems across the bank.
- No copy/paste wording across adjacent questions (rephrase the clinical setup).
- Use standard clinical terms (avoid slang, avoid vague “it/this”).

## 7) Content boundaries
- Educational only (no real patient identifiers).
- No step-by-step instructions for unsafe procedures beyond standard nursing awareness.
